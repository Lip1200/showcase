# ramosf.tech — Portfolio

A personal portfolio site built with Next.js.

## Getting Started

### Install dependencies
```bash
npm install
```

### Start dev server
```bash
npm run dev
```

### Build for production
```bash
npm run build
npm start
```

## Project Structure

- `app/` – routes & pages
- `components/` – UI building blocks  
- `public/` – static assets
- `styles/` – global styles
